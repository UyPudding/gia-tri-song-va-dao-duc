# Life Values and Ethics for Vietnamese High School Students

## Project Overview

This website is a **demo project** about **life values and moral education for Vietnamese high school students**.  
The project is created **for academic purposes only** and is **not an official website**.

## Purpose

- Introduce basic life values and ethics for high school students  
- Help students understand moral behavior and positive living values  
- Practice basic web development skills  
- Complete a school assignment and receive grades  

## Target Audience

- Vietnamese high school students  
- Teachers and classmates  

## Project Creator

- Created by a Vietnamese high school student  
- Developed as part of a school learning project  

## Features

- Educational content about life values and ethics  
- Simple and easy-to-use interface  
- Content suitable for high school students  

## Technologies Used

- HTML  
- CSS  
- JavaScript
